一个环世界mod，数字存储

这是一个存储管理 mod，提供了类似ae的存储方案。通过存储核心、接口和磁盘柜的组合，你可以建立一个高效的物品管理网络。

可能存在大量bug！

存储核心
- 基础容量: 100组物品
- 可扩展: 通过连接磁盘柜增加容量
- 网络命名: 可以为每个存储网络自定义名称
- 电力驱动: 需要电力才能运行
- 物品隐藏: 存储在核心中的物品不会渲染，提升性能
- 预留机制: 设置可调节预留物品，作为不适配的原版保留路径

磁盘柜
- 容量扩展: 每个磁盘柜提供额外的存储空间
- 灵活连接: 可以连接到任意存储核心
- 热插拔: 可以随时添加或移除磁盘柜

存储接口
- 远程连接: 可以连接到任意地图的存储核心
- 自动传送: 放入接口的物品会自动传送到核心

跨地图功能

终端芯片植入体
- 研究解锁: 完成终端植入体研究
- 右键植入: 选择殖民者右键植入终端芯片
- 机械体支持: 给机械师植入后，其操纵的所有机械体自动获得能力

材料访问
植入终端芯片的殖民者和机械体可以：
工作台制作: 直接从核心中取材料进行制作
建造: 建造时可以直接使用核心中的材料
材料统计: ResourceCounter 会统计所有地图核心中的材料
自动传送: 材料会自动传送到殖民者位置，无需手动搬运
据点交易适配：现在远行队出门不需要带东西了，只要队伍里有人有芯片就行
